# 🏢 Employee Management System (HRMS)

A comprehensive Human Resource Management System built with Python and Tkinter.

## 📋 Overview

This is a complete HR management system that helps organizations manage employees, attendance, leaves, payroll, and generate reports.

## ✨ Features

### Core Modules
- **👥 Employee Management** - Add, edit, delete, search employees
- **🏢 Department Management** - Manage company departments
- **⏰ Attendance Tracking** - Check-in/Check-out system
- **📅 Leave Management** - Request, approve, reject leaves
- **💰 Payroll Processing** - Calculate salaries automatically
- **📊 Reports** - PDF reports, Excel export, charts
- **⚙️ Settings** - Customize application behavior
- **👥 User Management** - Add, edit, delete users (Admin only)

### Advanced Features
- 🌙 **Dark/Light/Blue Theme** support
- 📧 **Email Notifications** (for leaves, payroll, new employees)
- 💾 **Database Backup & Restore**
- 📄 **PDF Report Generation**
- 📊 **Statistical Charts**
- 🔐 **Role-Based Access Control**

## 🛠️ Technologies Used

| Technology | Purpose |
|------------|---------|
| Python 3.8+ | Core programming language |
| Tkinter | GUI framework |
| SQLite3 | Database |
| ReportLab | PDF generation |
| Matplotlib | Charts and graphs |
| Pandas | Excel export |
| SMTP | Email notifications |
